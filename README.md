### DESKRIPSI

RakDigital adalah website aplikasi perpustakaan digital yang dibangun untuk mempermudah pengguna (user dan admin) dalam mengakses, mencari, dan mengelola koleksi buku secara online. Website ini bersifat dinamis, terhubung ke database, dan terdiri dari fitur-fitur yang sesuai dengan kebutuhan client.

---

### FITUR USER

1. Registrasi akun (menyimpan username dan password)
2. Login ke sistem
3. Melihat daftar buku
4. Mencari buku berdasarkan:
   - Judul
   - Penulis
   - Tahun terbit
5. Memfilter buku berdasarkan:
   - Kategori
   - Tahun terbit
6. Melihat detail buku
7. Meminjam buku (jika sudah login)
8. Melihat profil dan riwayat peminjaman
9. Edit profil - (Maaf, Fitur Belum Tersedia)
10. Logout

---

### FITUR ADMIN

1. Login khusus via `/admin/index.php`
2. Melihat dashboard statistik:
   - Jumlah buku
   - Jumlah anggota
   - Buku yang Tersedia
   - Buku yang sedang dipinjam
3. Kelola data buku:
   - Tambah
   - Ubah
   - Hapus
   - Filter berdasarkan kategori
4. Kelola data user (anggota):
   - Lihat daftar anggota
   - Edit dan hapus data anggota
   - Cari berdasarkan nama/email
